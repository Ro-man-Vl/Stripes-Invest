Заменяем имя папки на имя проекта
Сборка не содержит «node_modules». Для её развёртывания выполняем команду «npm i»
Удаляем ненужные задачи
Удаляем существующую папку "git"
Создаём новый репозиторий на «github»
Инициализируем git в папке проекта, следуя инструкциям из «gitReadme.txt»
Запускаем в режиме разработки «gulp»
Для формирования конечных файлов запустить в режиме «gulp --production»
